# etheramid
Decentralised social pyramid

### Multi-level pyramid.

### Send 1 Ether to enter.

### Invite other participants and get ether back from each pay:
* 1st level - 50%
* 2nd level - 25%
* 3rd level - 12.5%
* 4th level - 6.25%
* 5th level - 3.125%
* 6th level - 1.5625%
* 7th level - 1.5625%

Or just wait until random selects you for new perticicant without inviter.
